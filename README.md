# Advanced Calculator 🧮

A responsive web-based Advanced Calculator built using HTML, CSS, and JavaScript. The calculator supports both basic arithmetic operations and scientific calculations with a modern user interface and calculation history feature.

## Features ✨

* Basic Arithmetic Operations (+, −, ×, ÷)
* Scientific Functions (sin, cos, tan, log, ln, √, x², 1/x, π, e)
* Calculation History
* Keyboard Support
* Responsive Design
* Dark Theme Interface
* Error Handling for Invalid Expressions

## Technologies Used 🛠️

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run 🚀

1. Download or clone the repository.
2. Open the `index.html` file in any modern web browser.
3. Start performing calculations.
